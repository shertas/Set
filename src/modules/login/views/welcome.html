Bienvenido a la web de Proyecto SET

Instrucciones

Encontrar la mayor cantidad de sets. Un set es un grupo de tres cartas donde cada característica, considerada por separado, es toda igual o toda diferente.

Características de las cartas

  • Número: 1, 2 o 3 figuras
  • Forma: óvalo, rombo o curva
  • Color: rojo, verde o morado
  • Relleno: vacío, rayado o sólido

Regla principal

Para formar un set, cada característica debe ser:

  • Toda igual entre las tres cartas, o
  • Toda diferente entre las tres cartas.

Si una característica coincide solo en dos cartas, no es un set.

Fin del juego

El juego termina cuando se acaba el mazo y ya no queda ningún set. Gana quien tenga más sets reunidos.
